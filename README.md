# ✝️ Semana Santa de Jerez y Málaga

Repositorio dedicado a recopilar información sobre la **Semana Santa de Jerez de la Frontera** y la **Semana Santa de Málaga**, reuniendo datos históricos, curiosidades, fotografías y todo aquello que forma parte de este importante patrimonio cultural andaluz.

## 📖 Descripción

El objetivo de este proyecto es crear una fuente de información organizada y de fácil consulta sobre las hermandades, cofradías y todos los elementos que rodean la Semana Santa de ambas ciudades.

La información se encuentra estructurada para facilitar su mantenimiento y ampliación, convirtiéndose en una base de datos en constante crecimiento.

## 📚 Contenido

En este repositorio podrás encontrar:

- 📅 Información histórica.
- ⛪ Hermandades y cofradías.
- 👑 Imágenes titulares.
- 📍 Templos y sedes canónicas.
- 🗺️ Recorridos e itinerarios.
- 📸 Fotografías.
- 🎺 Bandas y acompañamientos musicales.
- 🎼 Marchas procesionales.
- 💡 Curiosidades y anécdotas.
- 📜 Efemérides.
- 📖 Documentación y referencias.

## 🎯 Objetivo

Este proyecto nace con la intención de preservar, organizar y compartir información sobre la Semana Santa de Jerez y Málaga, ofreciendo un recurso útil tanto para aficionados como para investigadores y desarrolladores.
